How to Fix ‘The following Components are required to run this program - DirectX Runtime’ Error

Many PC games require DirectX to run. If you see an error like “The following Components are required to run this program - DirectX Runtime,” follow these steps to fix it:

1. Download and Install the Latest DirectX

  • Visit the official Microsoft DirectX End-User Runtime Web Installer page
  • Download and run the installer
  • Follow the prompts to complete installation

2. Update Windows

  • Go to Settings > Update & Security > Windows Update
  • Check for updates and install all available updates

3. Reinstall the Game

  • Sometimes, reinstalling the game will automatically install the required DirectX components

4. Check for Missing DLLs

  • Use FixMissingDLL.com to scan and repair missing DirectX DLL files automatically

5. Run as Administrator

  • Right-click the game shortcut > Properties > Compatibility > Run as administrator

Tip: Always download DirectX from the official Microsoft website to avoid malware.
