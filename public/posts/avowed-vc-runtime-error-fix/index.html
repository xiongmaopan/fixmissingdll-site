How to Fix Avowed Microsoft Visual C++ Runtime Error on PC

Are you getting a Microsoft Visual C++ Runtime Error when launching Avowed or other PC games? This is a common issue caused by missing or corrupted Visual C++ components. Here’s how to fix it:

1. Use Steamworks Common Redistributables

Steam automatically downloads essential redistributables like DirectX and Visual C++ for each game. To manually install them:

  • Open Steam > Library > search for “Steamworks Common Redistributables”
  • Right-click > Manage > Browse local files
  • Go to _CommonRedist/vcredist, run all VC_redist installers (x64 and x86)
  • Restart your PC and try launching the game again

2. Reinstall or Repair Visual C++ Redistributables

  • Press Windows + R, type appwiz.cpl, press Enter
  • Find all Microsoft Visual C++ Redistributable entries
  • Right-click each, select Change > Repair
  • If the error persists, uninstall all, then download the latest versions from Microsoft and reinstall

3. Update Graphics Drivers

  • Outdated GPU drivers can cause runtime errors
  • Update via Device Manager or your GPU manufacturer’s website

4. Verify Game Files

  • In Steam, right-click the game > Properties > Installed files > Verify integrity of game files

5. Run as Administrator

  • Right-click the game shortcut > Properties > Compatibility > Run as administrator
